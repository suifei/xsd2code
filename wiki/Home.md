## 🎯 关于XSD2Code

XSD2Code 是一个强大的命令行工具，专门用于将 XML Schema Definition (XSD) 文件转换为多种编程语言的类型定义和数据结构。

### ✨ 核心特性

- **🌍 多语言支持** - Go, Java, C#, TypeScript, Python
- **🎛️ 智能处理** - 自动检测XSD特性，智能生成代码
- **📋 完整约束** - 全面支持XSD约束验证
- **⚡ 零配置** - 开箱即用，无需复杂配置
- **🔄 自动化** - 完美集成CI/CD流程

### 🏷️ 版本信息

- **当前版本**: v3.1.0
- **发布日期**: 2025年6月2日
- **许可证**: MIT License
- **语言**: Go 1.21+
